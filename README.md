# 임밍아웃 로또 🎰👶

로또 번호를 생성하다가 갑자기 초음파 영상으로 바뀌는 특별한 프로젝트!

## 🚀 특징

- React + TypeScript + Chakra UI로 만들어진 현대적인 웹 앱
- 로또 번호 생성 기능
- 5번 사용하면 초음파 영상 서프라이즈! 👶
- GitHub Pages에 배포 가능

## 🛠️ 기술 스택

- **Frontend**: React 18, TypeScript
- **UI Library**: Chakra UI
- **Build Tool**: Vite
- **Deployment**: GitHub Pages

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# GitHub Pages 배포
npm run deploy
```

## 🎯 사용법

1. "번호 생성하기" 버튼을 클릭하세요
2. 로또 번호가 생성됩니다
3. 5번 정도 반복해보세요...
4. 갑자기 초음파 영상이 나타납니다! 👶
